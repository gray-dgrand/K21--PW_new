# Bài tập buổi 2 - Key takeaways

## Git
- `working directory`: thư mục dự án nơi các file đang được chỉnh sửa (chưa chắc đã sẵn sàng commit).
- `staging area` (bộ nhớ đệm): nơi ta đưa các thay đổi vào để chuẩn bị commit.
- `repository`: nơi lưu các commit đã được ghi lại (dạng lịch sử thay đổi).

## JavaScript
- Toán tử chia dư `%` dùng để lấy phần dư, ví dụ: `a % b`.
- Điều kiện (`if/else`) để kiểm tra giá trị và quyết định chạy lệnh tương ứng.
- `for` để lặp theo phạm vi (bắt đầu -> kết thúc -> bước nhảy).
- Nối chuỗi với biến bằng toán tử `+`, ví dụ: `console.log("Số: " + n)`.
- Có thể in kết quả bằng `console.log(...)`.
