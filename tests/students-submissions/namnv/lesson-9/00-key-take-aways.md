# Lesson 9 - Key takeaways

## Vim
- `i` de vao insert mode.
- `ESC` de thoat insert mode.
- `:wq` de luu va thoat.
- `:q!` de thoat khong luu.

## Git
- `git merge <branch>` de hop nhat nhanh.
- `git rebase <branch>` de doi base commit va giu lich su gon hon.
- Sau khi rebase/squash tren nhanh da push, co the can `git push -f` (can than vi ghi de lich su remote).
- Fast-forward merge xay ra khi nhanh dich chua co commit rieng.
- Three-way merge xay ra khi hai nhanh da diverge.

## Playwright
- Co the viet test voi CSS selector (`locator('#id')`, `locator('.class')`) hoac Playwright locator (`getByRole`, `getByLabel`).
- Tach test theo file/module de de maintain.
- Dat ten test theo `case code + case name` de trace testcase de hon.
