# Bài tập buổi 3 - Key takeaways

## Git
- `revision` (commit hash) là mã định danh của mỗi commit.
- Dùng `git checkout <revision>` để xem lại code ở một mốc commit cũ.
- `git commit --amend` để chỉnh commit gần nhất.
- `git restore --staged <file>` để bỏ file khỏi staging.
- `git reset HEAD~1` để quay về commit trước đó (lùi 1 commit).

## JavaScript
- Object có thể chứa thuộc tính lồng nhau (nested object).
- Có thể thêm/xóa/sửa thuộc tính object bằng cú pháp chấm.
- Vòng lặp `for` xử lý dãy số, bảng cửu chương, tạo mảng.
- Hàm (`function`) nhận tham số và trả về kết quả.
