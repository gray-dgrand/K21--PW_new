# Lesson 03

- Git nâng cao với `git commit --amend`, `git restore --staged`, `git reset HEAD~1`.
- Hiểu revision/commit hash và cách checkout về mốc commit cũ.
- Làm việc với object: truy cập thuộc tính lồng nhau, thêm/xóa thuộc tính.
- Viết hàm có tham số, trả về kết quả và xử lý mảng bằng vòng lặp.
