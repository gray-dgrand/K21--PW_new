# Lesson 6 - Key Takeaways

## Git workflow
- Luon cap nhat nhanh chinh (`main`/`master`) truoc khi tao nhanh moi.
- Dat ten nhanh theo convention de de review.
- Day nhanh len remote, tao PR, gan reviewer va assignee.
- Xu ly comment review bang cach fix code, push lai va resolve thread.

## Code review
- Comment dung trong tam vao logic, naming, test va maintainability.
- Neu code on, phan hoi ro rang va xac nhan da review.
- Neu can fix, ghi comment cu the de tac gia de tai hien van de.

## TypeScript
- Type annotation giup phat hien loi som.
- `interface` dinh nghia hop dong du lieu cho object.
- `class` to chuc du lieu + hanh vi, de mo rong va bao tri.
