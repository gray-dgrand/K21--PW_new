# Lesson 7 - Key takeaways

## Live Server
- Dung extension Live Server de mo nhanh file HTML va reload tu dong.
- URL local thuong la `http://127.0.0.1:5500`.

## DOM va Selector
- Hieu cau truc DOM tree: parent, child, sibling, ancestor, descendant.
- Chon phan tu bang XPath tuyet doi va XPath tuong doi.
- Dung `text()` de match text chinh xac.
- Dung `contains(text(), "...")` de match text linh hoat khi co khoang trang hoac noi dung thay doi.

## Thuc hanh test automation
- Phan tich DOM giup viet locator dung va on dinh hon.
- Uu tien selector ro nghia, de doc, de bao tri.
