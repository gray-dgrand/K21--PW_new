# Lesson 8 - Key takeaways

## Git
- Nen thong nhat nhanh mac dinh la `main` cho cac repo moi.
- Dat ten nhanh theo convention de de track va review.
- Tach nhanh key-takeaways va nhanh bai tap code de PR ro rang hon.

## Playwright test design
- To chuc test theo module, moi module mot file `.spec.ts`.
- Dat ten `describe` theo format: `${module-code} - ${module-name}`.
- Dat ten test theo format: `${case-code}: ${case-name}`.
- Dung `test.beforeEach` de giam lap code.
- Dung `test.step` de chia nho hanh dong va de doc report.

## Data-driven mindset
- Lay du lieu login/account tu bang testcase de viet test on dinh.
- Moi expected can co assert ro rang, tranh assert qua chung chung.
