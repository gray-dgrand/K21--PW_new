# Lesson 10 - Key takeaways

## Extends và constructor
- `extends` giup class con tai su dung thuoc tinh/phuong thuc tu class cha.
- `constructor` chay khi khoi tao object moi.
- Trong class con, dung `super(...)` de goi constructor cua class cha.

## Export / Import
- Dung `export` de xuat class/function/const sang file khac.
- Dung `import` de su dung lai code va tach file ro rang hon.
- Duong dan import co the bo duoi `.ts` trong TypeScript.

## POM (Page Object Model)
- Tach locator + action vao class page de test gon hon.
- Test file chi mo ta flow, con thao tac UI dat trong page object.
- De refactor: giam lap code, de maintain, de review.

## Refactoring
- Thuc hien khi them tinh nang, review code, handover.
- Dau hieu can refactor: method qua dai, qua nhieu parameter, class qua lon.
